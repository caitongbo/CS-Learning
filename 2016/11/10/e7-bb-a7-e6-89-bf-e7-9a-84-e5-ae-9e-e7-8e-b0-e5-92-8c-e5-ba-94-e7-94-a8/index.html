Java 继承的实现和应用

1. （1）定义一个球类Ball，包含一个私有成员变量——半径(double r)，两个公有成员方法：设定半径方法（void setR(double x)）、得到半径值方法（double getR()）。 （2）定义一个台球类Billiards，继承Ball类，包含一个私有成员变量——颜色（String color），两个公有成员方法：设定颜色方法setColor()、输出信息方法show()，其中show()方法可以输出该台球的颜色和半径值。 （3）定义测试类Test输出台球类的主要信息。

class Ball{ //定义球类Ball
private double doubler;
public Ball(double doubler){
this.doubler=doubler;
}
public void setR(double doubler){ //设置半径大小r
this.doubler=doubler;
}
public double getR(double doubler){ //输出半径大小r
this.doubler=doubler;
return doubler;
}
}
class Billiards extends Ball{
public Billiards(double doubler) {
super(doubler); // TODO Auto-generated constructor stub
}
String color;
public void setColor(String color){
this.color=color;
}
public void show(double x,String y){
double doubler=x;
String color=y;
System.out.println(“台球的半径是：”+doubler);
System.out.println(“台球的颜色是：”+color);
}
}
public class EXP9_1 {
public static void main(String args[]){
Ball Ball_1=new Ball(10);
System.out.println(“球的半径：”+Ball_1.getR(10));
Billiards Billiards_1=new Billiards(10);
Billiards_1.show(10, “red”);
}
}

2． （1）根据下面的要求实现圆类Circle。 ① 圆类Circle的成员变量：radius表示圆的半径。 ② 圆类Circle的方法成员： Circle()：构造方法，将半径置0 Circle（double r）：构造方法，创建Circle对象时将半径初始化为r double getRadius()：获得圆的半径值 double getPerimeter()：获得圆的周长 void disp()：将圆的半径、圆的周长和圆的面积输出到屏幕 （2）继承第（1）题中的圆Circle类，派生圆柱体类Cylinder。要求如下： ① 圆柱体类Cylinder的成员变量：height表示圆柱体的高。 ② 圆柱体类Cylinder的成员方法： Cylinder（double r，double h）构造方法，创建Cylinder对象时将圆半径初始化为r，圆柱高初始化为为h double getHeight()：获得圆柱体的高 double getVol()：获得圆柱体的体积 void dispVol()：将圆柱体的体积输出到屏幕 （3）按照第（2）题的Cylinder类，接上键盘的输入值作为圆的半径和圆柱体的高，计算圆柱体的体积。

class Circle{
double radius;
double pi=3.14;
public Circle(){
this.radius=0;
}
public Circle(double r){
this.radius=r;
}
public double getRadius(){ //获取圆的半径
return radius;
}
public double getPerimeter(){ //求圆的周长
return 2*pi*radius;
}
public void disp(){
System.out.println(“圆的半径：”+getRadius());
System.out.println(“圆的周长：”+getPerimeter());
System.out.println(“圆的面积：”+pi*radius*radius);
}
}
class Cylinder extends Circle{
double height;
public void Cylinder(double r,double h){
radius=r;
height=h;
}
double getHeight(){ //获取圆柱体的高
return height;
}
double getVol(){ //求圆柱体的体积
return pi*radius*height;
}
public void dispVol(){
System.out.println(“圆柱体的高：”+getHeight());
System.out.println(“圆柱体的体积：”+getVol());
}
}
public class EXP9_2 {
public static void main(String args[]){
Circle C1=new Circle(10);
C1.disp();
Cylinder Cyl_1=new Cylinder();
Cyl_1.radius=8;
Cyl_1.height=5;
Cyl_1.dispVol();
}
}
