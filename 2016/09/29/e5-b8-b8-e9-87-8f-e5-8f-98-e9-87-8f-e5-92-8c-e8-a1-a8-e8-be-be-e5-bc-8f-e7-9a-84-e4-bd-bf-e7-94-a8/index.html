1. 编写一个java程序，该程序声明各种基本数据类型的变量，对它们初始化，再显示它们各自的值。

public class UseVariable {
public static void main(String args[])
{
boolean a=true;
short b=34;
int c=9;
long d=190;
double e=3.13432;
char f=’K’;
float g=12.6f;
String h=”完美”;
System.out.println(“boolean a=”+a);
System.out.println(“short b=”+b);
System.out.println(“int c=”+c);
System.out.println(“long d=”+d);
System.out.println(“double e=”+e);
System.out.println(“char f=”+f);
System.out.println(“float g=”+g);
System.out.println(“string h=”+h);

}

}

2. 输入并运行下面的程序并分析结果，体会各种表达式的使用方法，并回答问题。

public class E2_1{
public static void main(String args[]){
final double PI=3.14159;
int a=5,b=3,c=0;
double d=2.5;
System.out.println(“c=”+(a+b+a%b));
System.out.println(“c=”+a+b+a%b);
d=c*b+a/d*b;
System.out.println(“d=”+d);
}
}

public class E2_2{
public static void main(String args[]){
final double PI=3.14159;
int a,b,c;
a=b=c=5;
a=++b-++c;
System.out.println(“a=”+a+”b=”+b+”c=”+c);
a=b+++c++;
System.out.println(“a=”+a+”b=”+b+”c=”+c);
a=b–+c–;
System.out.println(“a=”+a+”b=”+b+”c=”+c);
}
}

public class E2_3{
public static void main(String args[]){
boolean x,y;
x=5>3||3*4<6&&!(5>3);
y=2==6&&9>8+4||5+6<+4&&!x;
System.out.println(“x=”+x+”,y=”+y);
}
}

(1)上述三个程序中各用到了哪些运算符？运算符的优先级如何？ 加法(+)、乘法()、除法(/)、求余(%) 乘除优先于加减，求余优先于加减; 自加自减(++、–)、加法(+); 关系运算符(> 、<)、乘法()、加法(+)、逻辑运算符(||、 &&) 算数运算优于关系优于逻辑运算符。 (2)增1和减1运算符（++和–）放在变量之前和放在变量之后，其用法有何不同？ 放在之前，自变量先增1或者先减1,之后参与运算 3.输入并运行下面的程序并分析结果。

public class E3{
public static void main(String args[]){
byte b=10;
char c=’a’;
int i=90;
long l=555l;
float f=3.5f;
double d=1.234;
float f1=f*b;
int i1=c+i;
long l1=l+i1;
double d1=f1/i1-d;
System.out.println(“f1=”+f1);
System.out.println(“i1=”+i1);
System.out.println(“l1=”+l1);
System.out.println(“d1=”+d1);
}
}

(1)上述程序用到了哪些自动类型转换？分别如何实现？ (2)何时需要用到强制类型转换？可能会造成什么不良后果？ (1)字节型->浮点型; 字符型->整形; 整形->长整型; 浮点型->双精度型; 所在数据元素的元素类型占有内存空间大小小于参与运算的其他数据类型，自动向高位转换，直到达到最高位结束。 (2)高位的数据类型向比其低位的数据类型转换需强制加转换符，否则容易导致数据丢失或出错。
